^【經】舍利弗！有菩薩摩訶薩行六波羅蜜時，成就三十二相，諸根淨利。諸根淨利故，眾人愛敬；以愛敬故，漸以三乘法而度脫之。如是，舍利弗！菩薩摩訶薩行般若波羅蜜時，應學身清淨、口清淨。

是菩薩欲令眾生眼見其身得度故，以三十二相莊嚴身。

「諸根淨利」者，眼等諸根明利，出過餘人；信、慧根諸心數根等，利淨第一；見者歎其希有：「我無此事！」愛敬是菩薩，信受其語，世世具足道法，以三乘道入涅槃。
